# Space War

Space War is a 2D top-down shooter game developed in Python with Pygame.

## Features

- Player movement with WASD keys.
- Player rotation with the mouse.
- Shooting with the left mouse button.
- Different types of enemies with different shooting patterns.
- Waves of enemies with increasing difficulty.
- Options menu to change the resolution, toggle fullscreen, and adjust the FPS limit.
- Saving and loading of the game settings.

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/diegosarc/space-war.git
   ```
2. Install the dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the game:
   ```bash
   python space_war.py
   ```

## Usage

- Use the WASD keys to move the player.
- Use the mouse to aim and shoot.
- Press the escape key to exit the game.
